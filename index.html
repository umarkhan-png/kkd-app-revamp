किसान का अपना ऐप v0.2 visual reset

What changed from v0.1: dropped Partner App's dark gradient hero → warm cream #FAF7F2 base with sun-warm accents; bigger Devanagari script; light/warm palette throughout. Flow change: B2C browse-first — Language → Home (logged-out browse with ₹100 unlock banner) → Login is contextual, triggered when user wants the offer or hits a locked feature (Krishi Doctor, Cart). No more upfront forced signup.
