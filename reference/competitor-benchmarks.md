## 1. DeHaat (DeHaat Kisan: Farming Guide) — *agrevolution.in*

**Positioning:** Full-stack agri platform — advisory + commerce + finance + insurance + soil testing. Built around 8,000+ physical DeHaat Centers (omnichannel).

### Modules in app
| Module | Detail |
|---|---|
| Crop advisory | Farm tagging, expert agronomist guidance, pest/health/yield monitoring |
| Mandi rates | Region-wise local mandi rates, direct institutional buyer connections |
| Weather | Daily + 14-day forecast on agri parameters |
| Soil testing | Lab-tested mineral availability → fertilizer recommendation |
| Inputs commerce | Home + field delivery; **payment-at-delivery (COD)** + pickup at center |
| Agri finance | Credit + installment programs for equipment |
| Crop insurance | Parametric weather-based, satellite-driven |
| Community | Posts, expert Q&A, articles, videos, agri-news, **quiz with prizes** |
| Voice | **Voice calls in regional languages** — actual human-assisted, not just voice input |
| Premium membership | Bundled annual plan |

### Languages
**6+ regional languages**, plus voice-call support in regional dialects (human, not TTS).

### What to steal
- **Voice calls in regional language** — beyond voice search, actual human help is the trust unlock for non-literate users.
- **Quiz on community tab** with prizes — engagement hook + product education combined.
- **Farm tagging** as identity — single-place to capture crop + plot info, used across advisory/insurance/recommendations.
- **Omnichannel delivery options** — COD-at-door OR pickup-at-center. KKD's 85% COD base mirrors this need.
- **Premium membership bundle** — single SKU that aggregates value across modules; raises LTV.

### What to skip / KKD-specific gaps
- Finance + insurance are deep verticals — out of KKD V1 scope.
- Soil testing requires physical infra DeHaat has and KKD doesn't.

## 2. AgroStar (Kisan Agridoctor App) — *corporate.agrostar.in*

**Positioning:** Advisory-first with 150+ brand catalog. Three-touchpoint omnichannel: app + advisory call center + Saathi Stores.

### Modules in app
| Module | Detail |
|---|---|
| **Agri Doctor** (hero module) | Personalized agronomy advice from experts |
| Crop info | 50+ crops × disease/nutrition guidance |
| Inputs commerce | 150+ brands, 200+ products |
| Community | Post queries, peer + expert answers, instructional videos |
| Weather | Location-based rainfall/temp/humidity |
| Brand catalog | Premium-brand-led merchandising |

### Languages
**11 regional languages** — Hindi, Gujarati, Marathi, English, Telugu, Kannada, Punjabi, etc. **Widest coverage in the set.**

### What to steal
- **"Agri Doctor"** as a single named entry point for advisory — sticky brand for the help feature. KKD currently has nothing like this.
- **11-language coverage** — V2 expansion target. V1 covers 95% with 5; v2 add Kannada/Odia/Tamil for full match.
- **Brand-led catalog UX** — 150+ brand surface gives farmers a recognition shortcut ("I trust X brand"). KKD only has Katyayani brand, so the analogue is **product-family/sub-brand surfacing** (Chakrawarti, Chakraveer, Antivirus as sub-brands).
- **Tri-touchpoint loop** — App → call center → physical store. KKD has no physical retail, so app → call center → home delivery is the analogue. Make the in-app "call us" CTA prominent.

### What to skip
- Multi-brand catalog is AgroStar's moat. KKD is single-brand — don't pretend otherwise; lean into "official Katyayani app" trust positioning.

## 3. BigHaat (Kisan Agri Farming App)

**Positioning:** Pure-play agri inputs e-commerce. 9,000+ SKUs, AI crop diagnosis, fertilizer calculator. **Highest rating (4.6) in the set** — the e-commerce UX bar to beat.

### Modules in app
| Module | Detail |
|---|---|
| Inputs commerce | **9,000+ SKUs** — seeds, fertilizers, crop protection, implements; national + intl brands |
| **AI crop health diagnosis** | Upload plant photo → AI returns disease + solution + product recommendation |
| **Fertilizer Calculator** | Crop + acreage → exact organic fertilizer quantity → reduces waste |
| Mandi prices | Real-time, region-specific |
| Weather | Customized agro-weather forecast |
| **Kisan Vedika** (community) | Farmer-to-farmer network, posts, advisories |
| Payments | **UPI, wallets, credit card, debit card** — full digital stack, NOT just COD |
| Brand surface | Strong brand merchandising |

### Languages
**5 languages** — English, Hindi, Tamil, Telugu, Kannada. Same V1 footprint KKD is targeting (minus Bengali/Marathi).

### What to steal
- **AI photo diagnosis** — single best feature in the set. Photo → diagnosis → recommended product = closed-loop conversion. Direct unlock for KKD's "discovery is broken" pain.
- **Calculator tools** — fertilizer calculator (BigHaat) + profit calculator analogue. Useful + sticky; drives upsell (right quantity = bigger order).
- **Full digital payment stack** — UPI prominent, not buried. KKD has UPI in scope but 85% COD; surfacing UPI better is the prepaid-push play (Katyayani Parichay offer).
- **4.6-rating UX** — closest e-commerce model to KKD. Use as visual + interaction reference for ATC, cart, checkout.
- **National + intl brand callout** — trust signal. KKD analogue: certifications, FCO/CIB-CR licence numbers, lab-test badges.

### What to skip
- 9,000-SKU breadth is BigHaat's moat. KKD ships ~17 SKUs; lean into curated/expert-picked positioning instead of catalog-depth race.

## 4. Krishify (Farmer Community App)

**Positioning:** Social-first — "Facebook for Indian farmers". 90L+ users; community-led growth, not commerce-led.

### Modules in app
| Module | Detail |
|---|---|
| **Community feed** (hero) | 25+ digital communities, 2k+ members each, peer + expert posts |
| **Voice search** | Vernacular voice query (across feed, products, advisories) |
| Crop doctor | Free Q&A with experts |
| Peer-to-peer trade | Sell cattle, tractors, used equipment farmer-to-farmer |
| Mandi rates | Real-time |
| Weather | Real-time |
| **Smart Plots** | Satellite + remote-sensing monitoring |
| Video / UGC | Creator monetization for video uploads |

### Languages
Multi-lingual (count not surfaced on landing, but they've publicly emphasized voice search to bridge "linguistic / communication barriers" — implies wide vernacular support).

### What to steal
- **Voice search** — direct fix for KKD's #1 leak (search/discovery). Speech-to-text in Hindi/Bengali/Marathi/Telugu, fuzzy match to SKUs + crop problems. **Highest ROI single feature.**
- **Community feed at home** — but KKD shouldn't try to be Krishify. Instead use **lightweight "Farmer Stories" or "Reviews with photos"** strip on home + PDP — social proof, not full social network.
- **Crop-locality communities** — group buying / referral seed unlock. Out of V1 but worth a "post-conversion-fix" wave plan note.

### What to skip
- Full social-network features (feed, follow, DMs) — not KKD's job. Commerce conversion is the fire.
- Smart Plots / satellite — premium feature, out of V1.
- Creator monetization — distraction.

## Cross-cutting patterns farmers expect (must-have parity)

Every app in the set has these. If KKD doesn't, it feels broken:

1. **Mandi rates / regional prices** — 4 of 4. Even if KKD is selling inputs (not produce), having mandi rates raises perceived value of opening the app daily.
2. **Weather forecast** — 4 of 4. Daily checkable surface = stickiness.
3. **Crop advisory / Ask-an-expert** — 4 of 4. Free Q&A or call-center.
4. **Vernacular UI** — 4 of 4. **5 langs is table-stakes**, 11 is best-in-class.
5. **COD payment** — at least DeHaat + BigHaat ship it. KKD's 85% COD share matches.
6. **Community / social proof** — 4 of 4 in some form. KKD has 0.
7. **Video content for crops** — 3 of 4. KKD has 0.
8. **Photo-based crop diagnosis** — BigHaat only. Differentiator.

## Where KKD can WIN (differentiation bets)

| Bet | Why winnable | Cost |
|---|---|---|
| **Voice-first search + ordering** | Krishify validates voice; nobody owns voice-to-cart end-to-end | Medium (STT + NLP) |
| **AI photo diagnosis → direct ADD to cart** | BigHaat does diagnosis, but route-to-purchase is weak. KKD can close the loop with single-brand inventory | High but moat-forming |
| **Expert hand-holding** (call-back + chat with Katyayani agronomist) | DeHaat does voice calls but generic; KKD can offer **product-specific** Katyayani agronomist callback | Medium (call-center ops) |
| **Curated kit / bundle for crop + problem** | "Tomato + leaf-curl kit" = pesticide + bio-stimulant + fertilizer auto-bundled, hits ₹2,500 Parichay prepaid threshold | Low (merchandising) |
| **Trust-led PDP** (lab certs, FCO licence, before/after farmer photos) | All competitors brand-trust generic; KKD as Katyayani's own app can over-index on trust | Low |
| **Same-day buy-readiness** (no tutorial walls) | Data says 43% same-day conv — competitors over-onboard. KKD can ship a "buy-ready home" | Low — pure design |

## IA conventions farmers already know (don't reinvent)

Common nav across the set:
- **Home / Feed** — hero merchandising + alerts
- **Shop / Products** — catalog + search
- **My Farm / My Crops** — saved crops + farm tagging
- **Community / Advisory** — Q&A + feed
- **Account / Orders** — profile + order history

→ **KKD V1 nav recommendation:** 5-tab bottom bar — `Home` · `Shop` · `My Crop` · `Advisory` · `Account`. Familiar to anyone coming from DeHaat / AgroStar / BigHaat. Voice-search FAB floats on Home + Shop.

→ Resolve internal naming clash (existing-app obs #15: "My farm" vs "My crop"): pick **"My Crop"** — matches AgroStar + DeHaat, simpler word for farmers.

## V1 must-haves derived from this benchmark

Must ship to be credible (table-stakes):
- 5-language UI with **language picker on first launch** (Hi default)
- Mandi rates (read-only feed, even if MVP)
- Weather forecast (location-based)
- Crop advisory: at minimum, "Call our agronomist" button + simple Q&A
- COD + UPI + partial payment
- Order tracking, address book, reorder
- Search (with voice as the differentiator)
- PDP with trust signals (license, lab cert, real farmer reviews)

Differentiation (one or two, not all):
- **Pick #1: Voice search end-to-end** (cheapest big win, fixes #1 data leak)
- **Pick #2: Photo crop-diagnosis → direct product** (highest moat)

Hold for V2:
- Full community feed
- Smart plots / satellite
- Insurance / finance modules
- Creator / UGC video
