KKD Design System

Katyayani Krishi Direct · farmer-tuned visual language · v1 · 2026-05-14

Distinct from Partner App. Built for farmers: bigger touch, larger icons, vernacular-first copy, voice-friendly entry points everywhere, trust-led PDPs (license + lab cert + factory-direct badge). Quality floor = Partner App level; KKD must exceed.

Principles

1 · Vernacular-first

Hindi (Devanagari) is the primary script on every screen. English sits below as smaller secondary. Hindi default on first launch — never auto-pick English.

2 · Voice-friendly

Mic FAB visible on Home + Shop + Search + Advisory. Tap-and-hold or single-tap. Direct-to-cart from voice. Biggest single win per data + Krishify validation.

3 · Buy-ready on first session

43% of all purchasers convert on day-0. No tutorial walls. Permissions deferred. Home shows price + ADD on hero SKUs immediately.

4 · Trust-led PDP

License + lab cert + factory-direct + batch expiry + real-farmer photos. Counters every K1–K5 review complaint.

5 · Bigger touch + icons

Minimum touch target 48px (vs 44 standard). Icons 28–32px on primary nav (vs 24 in Partner App). Body 14–15px minimum.

6 · Single splash, no loaders

Existing app has 4 loaders. KKD has ONE 2.5s splash → straight to language picker. No "Parichay splash", no homepage loader.

Anti-patterns (do NOT do)

  • ✗ 4 splash loaders chained — KKD has one
  • ✗ Permission popups before value comm — show what user gets first
  • ✗ Side popup that appears twice (Eng + Hindi) — single language-aware popup
  • ✗ Default-to-English language — Hindi default, picker first
  • ✗ Naming clash ("My farm" vs "My crop") — single canonical: My Crop
  • ✗ Naming clash ("Track order" vs "Order tracking") — single canonical: Track Order
  • ✗ 10-min delivery / lightning badges — agri ships in 5–7 days
  • ✗ Lifestyle photography — real Katyayani SKU photos only
  • ✗ Colorful filled icons — outline only, 2px stroke
  • ✗ Variant selector that eats vertical space — chip row + "see all" sheet
  • ✗ Rating math that double-counts delay + product — show breakdown
  • ✗ Signout without confirm — destructive confirm sheet
