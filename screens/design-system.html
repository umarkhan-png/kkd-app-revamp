19 · Core Patterns
Bottom-sheet modal
border-radius:24px 24px 0 0 · handle 40×4 #E2E8F0 · box-shadow:0 -8px 32px rgba(0,0,0,.18) · slide .3s cubic-bezier(.32,.72,0,1).
Sticky CTA footer
White bg, border-top:1px solid #F1F5F9, single emerald primary button, padding-bottom: calc(env(safe-area-inset-bottom,0px) + 12px).
Order / product tile
Outer card wraps an <a> (head → id → thumbs → divider → bottom); action row sits inside the same card, padded — never nested anchors, never a flush full-width strip.
Safe area
Use env(safe-area-inset-top/bottom, 0px) with the 0px fallback so the declaration survives on browsers without env().

22 · Developer Hand-off Notes
• Fonts: Poppins 400-900 (Google Fonts). Always set font-variant-numeric:tabular-nums on prices, quantities and IDs.
• Icons: Font Awesome 6.5.1 (CDN) for UI glyphs; illustrated category icons are inline 36×36 SVG (download above).
• Images: product cut-outs ship as *_nobg.png; crop thumbnails as JPG. Hover-download converts to WebP at native size.
• No gold/amber theme: amber is reserved for COD / in-transit / warnings only — never the brand colour.
• Weights: default 500-700; reserve 800-900 for one hero per section.
• Format on download: all assets export as WebP (q 0.92); SVGs rasterise at 256px on the long edge.
